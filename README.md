# Base_datos
•	Conclusiones finales sobre:
o	Consultas donde se guardan
Las consultas no se guardan, automáticamente solo es información a mostrar en base a las tablas propuestas 
o	rendimiento con índices
Los índices demostraron que se reduce considerablemente el tiempo de ejecución en consultas pesadas
o	utilidad de vistas
Las vistas fueron útiles ya que simplificaron consultas complejas y tambien facilitan el acceso a información sin exponer directamente la estructura de las tablas 
o	importancia de transacciones
Las transacciones son importantes ya que aseguran que múltiples operaciones se ejecuten con un solo bloque, manteniendo la integridad cuando ocurre un error inesperado 
o	uso de procedimientos
Los procedimientos almacenados fueron valiosos para automatizar operaciones complejas en un solo comando
